# UI Rules

- Premium quality reference: Apple, Linear, Raycast, Vercel, Toast POS, Square.
- Font: Inter for UI, refined display serif for restaurant brand moments.
- Base radius: 16px.
- Card radius: 24px.
- Spacing: 8px grid.
- Motion: 250ms ease.
- Icons: Lucide React.
- Shadows: soft and restrained.
- Theme: dark and light.
- Avoid generic admin templates.
- Avoid Bootstrap and Material visual language.
- Every screen must be portfolio-quality and commercially sellable.
